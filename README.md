Asset managers face extreme information overload from fragmented macroeconomic data, making it difficult to identify emerging regimes and regional contagion in real-time.

This project streamlines such complex information into a Predictive Intelligence Dashboard. It replaces manual monitoring with a systematic engine that synthesises data to provide an economic outlook from high-frequency indicators using a Random Forest Regressor to generate a Ghost GDP Nowcast. It utilises NumPy-based logic to calculate real-time Regime Probabilities (e.g., Stagflation, Tech Boom) by clipping and normalising indicator ratios, allowing managers to identify current economic themes. Additionally, it automatically classifies market regimes and visualises regional inflation and liquidity transmission risks across regions through a Plotly Geospatial Map, which uses conditional triggers to render transmission lines when inflation or liquidity stress exceeds specific thresholds across regions. Furthermore, it proposes specific risk implications for cross-asset classes based on current systemic stress. There are also systetmic stress triggers via Streamlit sidebar inputs, enabling managers to perform instantaneous portfolio stress-tests against hypothetical shocks without navigating multiple complex data sources. Current inputs are compared against past crises (eg. 1970s Great Inflation) by using Euclidean distance mapping to provide a Historical Similarity Score.
